# ⚽ FIFA 21 Player Market Analysis

A complete end-to-end **Data Analytics Project** built using **Python, Pandas, NumPy, Matplotlib, Seaborn, and Power BI**. This project focuses on cleaning raw FIFA 21 player data, performing Exploratory Data Analysis (EDA), creating meaningful visualizations, and generating business insights to support football club recruitment and transfer decisions.

---

# 📌 Project Overview

Football clubs invest millions of euros during every transfer window. Selecting the right player is one of the biggest challenges, as poor transfer decisions can result in financial losses and poor team performance.

This project analyzes the FIFA 21 player dataset to identify factors affecting player market value, discover hidden talent, analyze wage structures, and generate data-driven recommendations that support smarter recruitment decisions.

---

# 🎯 Business Problem

Football clubs require a systematic and data-driven approach to evaluate players before making transfer decisions. Traditional scouting methods may overlook valuable insights hidden within player statistics.

This project aims to answer important business questions such as:

* What factors influence a player's market value?
* Which young players have the highest future potential?
* Are clubs paying players fairly?
* Which players are undervalued?
* Which clubs are financially efficient?
* How can analytics improve recruitment decisions?

# 📂 Dataset Information

**Dataset:** FIFA 21 Raw Dataset

The dataset contains approximately **18,979 football players** with over **70 player attributes**, including:

* Player Name
* Age
* Club
* Nationality
* Best Position
* Overall Rating
* Potential Rating
* Market Value
* Wage
* Release Clause
* Height
* Weight
* Preferred Foot
* Skill Moves
* Weak Foot
* Contract Details
* Player Statistics

---

# 🧹 Data Cleaning

The following preprocessing steps were performed:

* Renamed abbreviated column names.
* Removed unnecessary columns.
* Handled missing values.
* Converted currency columns into numeric values.
* Cleaned Height and Weight columns.
* Converted Hits values into numeric format.
* Removed unwanted symbols from rating columns.
* Extracted contract expiry year.
* Trimmed extra whitespace.
* Created new business features.

### Feature Engineering

New features created:

* Potential Growth
* Wage to Value Ratio
* Skill per Euro
* Contract Year
* Number of Positions

# 💡 Key Insights

* Overall Rating is the strongest factor influencing Market Value.
* Young players with high Potential provide the best long-term investment.
* Better players generally receive higher wages.
* Several undervalued players offer excellent transfer opportunities.
* Certain clubs demonstrate greater financial efficiency.
* Wage-to-Value Ratio helps identify financially efficient players.
